# Tensorlake Cloud SDK

A Rust SDK for interacting with Tensorlake Cloud APIs.

This SDK provides Rust bindings to Tensorlake Cloud APIs. It's designed to be a building block for other projects that interact with Tensorlake Applications. With this SDK, you can deploy applications, manage secrets, create images, and trigger requests programatically.

For detailed documentation, see [docs.rs/tensorlake-cloud-sdk](https://docs.rs/tensorlake-cloud-sdk).
